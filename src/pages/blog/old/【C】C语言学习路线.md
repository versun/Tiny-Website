C： C Primer Plus, C和指针，C专家编程\

数据结构和算法：大话数据结构，剑指offer，刷题。\
计算机网络：TCP/IP详解\
操作系统：深入理解操作系统\
设计模式：大话设计模式\
应用与编程实践：\
Linux：鸟哥的Linux私房菜 或 Linux就该这么学\

Linux的环境编程：Unix环境高级编程 ，Linux高性能服务器编程，POSIX多线程程序设计\
《算法》第四版 红色\
